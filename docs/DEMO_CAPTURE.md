# Demo Capture Plan

Use this plan to produce launch-ready GIFs and short videos.

## Clip 1: Live Terminal

- Length: `20-40s`
- Flow:
  1. Open existing server profile
  2. Open terminal session
  3. Show live streaming output + ANSI colors
  4. Use terminal search next/prev

## Clip 2: AI + Fleet

- Length: `20-40s`
- Flow:
  1. Send shell command that triggers an error
  2. Tap AI suggestion / explain error
  3. Run fleet command across 2+ servers
  4. Show grouped success/error results

## Optional Clip 3: Glasses Mode

- Length: `15-30s`
- Flow:
  1. Enable glasses mode
  2. Start voice capture
  3. Stop + transcribe + send transcript

## Export Targets

- GIF previews for README (`docs/media/*.gif`)
- MP4 for social launch posts (X/Product Hunt/Reddit)
- App Store screenshots from the same sessions for message consistency
